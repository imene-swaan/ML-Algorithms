I-1-2- Normalisation¶

La normalisation est la mise echelle des valeurs des caractristiques. La standardisation Z-score peut tre considre comme un forme de normalisation. En appliquant Z-score sur une caractristique donne ($X_j$), la moyenne de ces valeurs sera 0 et l'cart-type sera 1.

Etant donne une matrice $X(M, N)$ avec $M$ chantillons et $N$ caractristiques.

  • $X_j$ est la caractristique $j$ ; c'est un vecteur de $M$ lments
  • $\mu$ est la fonction de moyenne
  • $\sigma$ est la fonction de l'crat-type
  • $Z-score$ est un vecteur de $M$ lments
$$Z-score(X_j)=\frac{X_j - \mu(X_j)}{\sigma(X_j)}$$

I-2- Descente du gradient¶

La formule pour estimer le rsultat est : $$\hat{Y} = \theta_0 + \sum\limits_{j=1}^{N} \theta_j X_j$$

Le but de l'apprentissage est de trouver les $\theta$ qui minimisent le taux d'erreur entre les rsultats $\hat{Y}$ estims et $Y$ rels.

I-2-3- Fonction du coût¶

La fonction du cot calcule l'erreur entre les valeurs estimes ($\hat{Y}$) et les valeurs relles ($Y$). Ici, on va utiliser l'erreur quadratique moyenne (MSE: mean square error).

On modifie un peu dans sa formule : on divise sur 2 (a reste toujours une fonction de cot). Etant donn un dataset avec $M$ chantillons:

$$J = \frac{MSE(Y, \hat{Y})}{2M} = \frac{1}{2M} \sum\limits_{i=1}^{M} (Y - \hat{Y})^2$$

I-2-4- Les gradients¶

Afin de mettre jour les paramtres ($\theta$), on doit distribuer l'erreur : chaque paramtre va tre mis jour avec une petite valeur de l'erreur.
Cette petite valeur est calcule en utilisant les dirives partielles.

$$\frac{\partial J}{\partial \theta_j} = \frac{\partial }{\partial \theta_j} \frac{1}{2M} \sum\limits_{i=1}^{M} (Y - \hat{Y})^2$$$$\frac{\partial J}{\partial \theta_j} = \frac{\partial }{\partial \theta_j} \frac{1}{2M} \sum\limits_{i=1}^{M} (Y - (\sum\limits_{k=0}^{N} \theta_k X_k))^2$$

O $X_0 = 1$ toujours $$\frac{\partial J}{\partial \theta_j} = \frac{1}{M} \sum\limits_{i=1}^{M} (Y^{(i)} - \hat{Y}^{(i)}) X_j^{(i)}$$ O $X^{(i)}$ est l'chantillon $i$ et $Y^{(i)}$ est sa classe

# TODO: Dfinir la fonction de gradient
# Elle doit retourner les gradients de tous les thtas
def gradient(X, Y, theta):
    error = (predire(X, theta)-Y)/X.shape[0]
    return  np.dot(X.T, error)

def gradient_descent(X, Y, theta, nb_iter=NB_ITER, learning_rate=LEARNING_RATE, affich = True):
    couts = []
    theta1 = theta.copy()
    
    if (affich):
        afficher_droite(X, Y, theta1)
    
    # Dfinir l'algorithme de la descente de gradient : 
    for i in range(nb_iter):
        couts.append(J(X, Y, theta1))
        grad = gradient(X, Y, theta1)
        theta1 -=  learning_rate * grad
        
        if affich and i == nb_iter/2:
            afficher_droite(X, Y, theta1)
        
    if(affich):
        afficher_droite(X, Y, theta1)

    return theta1, couts

# TODO : Dfinir la dscente du gradient
def gradient_descent2(X, Y, theta, nb_iter, learning_rate, affich = True, seuil=0.001):
    couts = []
    theta1 = theta.copy()
    couts.append(J(X, Y, theta1))
    
    if (affich):
        afficher_droite(X, Y, theta1)
    
    # Dfinir l'algorithme de la descente de gradient : 
    for i in range(nb_iter):
        grad = gradient(X, Y, theta1)
        theta1 -=  learning_rate * grad
        
        if affich and i == nb_iter/2:
            afficher_droite(X, Y, theta1)
        
        couts.append(J(X, Y, theta1))
        taux_changement = np.abs(couts[-1] - couts[-2])/couts[-2]
        if (taux_changement < seuil):
            break
        
    if(affich):
        afficher_droite(X, Y, theta1)

    return theta1, couts

III.6 Persistance des modèles¶

Aprs avoir entrain un modle, il est souhaitable de le conserver pour un usage ultrieur sans avoir besoin d'entrainer une deuxime fois. Il y a deux faons de le faire selon la doc de scikit-learn (https://scikit-learn.org/stable/modules/model_persistence.html)

  • la srialisation pickle
  • la srialisation joblib

La deuxime est recommande par scikit-learn. Aprs avoir entrainer notre modle, on le sauvegarde.

In [37]:
from joblib import dump, load

# Pour sauvegarder le modle
dump(regressor1, "mon_modele.joblib")

# Pour Rcuprer le modle
regressor_past = load("mon_modele.joblib")
