# une fonction qui normalise une matrice sur chaque colonne 
# dans l'entrainnement, on calcule les moyennes et les dviations de chaque colonne
# dans le teste, on ne doit pas recalculer ces deux paramtres. 
# Mais, on doit utiliser ceux calculs dans l'entrainnement
def normaliser(X, mean=None, std=None): 
    if (mean is None) or (std is None): 
        mean = np.mean(X, axis=0)
        std = np.std(X, axis=0)
    X_norm = (X - mean)/std
    return X_norm, mean, std

# Prparer la matrice des caractristiques pour l'entrainnement ou la prdiction
# On applique la normalisation sur chaque colonne (sauf si : norm=False)
# Ensuite on ajoute une colonne des 1 (sauf si : const=False)
def preparer(X, norm=True, const=True, mean=None, std=None): 
    X_pre = X.copy()
    if norm: 
        X_pre, mean, std = normaliser(X_pre, mean=mean, std=std)
    if const:
        X_pre = np.append(np.ones((X_pre.shape[0],1)), X_pre ,axis=1)
    return X_pre, mean, std

def ligne_decision(X, Theta, mean=[0, 0], std=[1, 1]):
    X1 = [np.min(X[:,0]), np.max(X[:,0])]
    X2_min = mean[1] - std[1] * (Theta[0] + Theta[1] * (X1[0] - mean[0])/std[0]) / Theta[2]
    X2_max = mean[1] - std[1] * (Theta[0] + Theta[1] * (X1[1] - mean[0])/std[0]) / Theta[2]
    return X1, [X2_min, X2_max]

I-2- Régression Ridge¶

I-2-1- La fonction du coût avec régularisation L2¶

La fonction du cot est donne par :

$$ cout(h_\theta(x), y) = - y \log(h_\theta(x)) - (1-y) \log(1 - h_\theta(x))$$$$J(\theta) = \frac{1}{m} \sum\limits_{i=1}^{m} cout(h_\theta(x^{(i)}), y^{(i)})$$

En appliquant une rgularisation L2 sur cette fonction, on aura :

$$J(\theta) = \frac{1}{m} \sum\limits_{i=1}^{m} cout(h_\theta(x^{(i)}), y^{(i)}) + \frac{\lambda }{2m} \sum\limits_{j=1}^{n} \theta_j^2$$

Remarque Dans les noncs, la rgularisation n'est pas divise par le nombre des chantillons $m$. Mais, en pratique, on la divise. Voici des rponses sur stackexchange expliquant pourquoi.

1-2-2- Les gradients avec régularisation L2¶

Les gradients sont calculs par: $$\frac{\partial J(\theta)}{\partial \theta_j} = \frac{1}{m} \sum\limits_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)}) x_{ij} $$

Voici les gradients rgularises avec L2 :
$$\frac{\partial J(\theta)}{\partial \theta_j} = \frac{1}{m} \sum\limits_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)}) x_{ij} + \frac{\lambda }{m} \theta_j$$

I-3-2- Entraînement : Iterative soft-thresholding algorithm (ISTA)¶

Le driv de $L1$ est : $$\frac{\partial L1}{\partial \theta_j} = \frac{\partial \frac{\lambda }{m} \sum\limits_{j=1}^{n} |\theta_j|}{\partial \theta_j} = \frac{\lambda }{m} \frac{\theta_j}{|\theta_j|}$$

Il est clair que le gradient n'est pas dfinit dans le cas $\theta_j = 0$. En appliquant le sous-diffrentiel, on aura : $$ \frac{\partial L1}{\partial \theta_j} = \begin{cases} -\frac{\lambda}{m} & \text{si } \theta_j < 0 \\ 0 & \text{si } \theta_j = 0 \\ \frac{\lambda}{m} & \text{si } \theta_j > 0 \\ \end{cases} $$

On ne peut pas utiliser la descente du gradient pour rsoudre a. On va utiliser un alorithme de descente du gradient approximative, qui s'appelle $ISTA$. $$prox(\theta) = S_{\lambda}(\theta) = \begin{cases} \theta_j + \lambda & \text{si } \theta_j < - \lambda \\ 0 & \text{si } -\lambda \le \theta_j \le \lambda \\ \theta_j - \lambda & \text{si } \theta_j > \lambda \\ \end{cases} $$

Vous devez implmenter cette fonction

Les thtas sont mises jour comme suite : $$ \theta = S_{\lambda}(\theta - \frac{\partial J_0}{\partial \theta})$$ O, $J_0$ est la fonction du cot sans rgularisation

Source : Ryan Tibshirani (2015), Lecture 08: Proximal Gradient Descent

class RegLogReg(object): # Rgression logistique rgularise
    
    def __init__(self, nbr_iter=100, alpha=.01, Lambda=0.01, norm=True, const=True, reg="l2"): 
        self.nbr_iter = nbr_iter
        self.alpha = alpha
        self.Lambda = Lambda
        self.norm = norm
        self.const = const
        self.reg = reg
    
    def entrainer(self, X, Y): 
        if self.reg == "l1":
            entrainer = entrainer_L1
        else:
            entrainer = entrainer_L2
            
        self.thetas, self.mean, self.std, self.couts = entrainer(X, Y, 
                                                                  nbr_iter=self.nbr_iter, 
                                                                  alpha=self.alpha, 
                                                                  Lambda=self.Lambda, 
                                                                  norm=self.norm, 
                                                                  const=self.const)
        
    # La prdiction
    # si prob=True elle rend un vecteur de probabilits
    # sinon elle rend une vecteur de 1 et 0
    def predire(self, X, prob=False, seuil=0.5, it=-1):
        H = h_1(X, self.thetas[it], self.mean, self.std, self.const)
        if prob:
            return H
        return (H >= seuil).astype(int)

II-1- Implémentation de ANOVA (one-way)¶

II-1-1- One way F value pour un seul attribue¶

Le nombre des chantillons pour chaque classes est diffrent. Pour cette raison, on va implmenter ANOVA for a CRD (COMPLETELY RANDOM DESIGN). On va implmenter l'algorithme expliqu dans Richard Horsly (2012) COMPLETELY RANDOM DESIGN (CRD). Pour valider l'implmentation, on va utiliser les mmes donnes de la page 10.

pour un attribut donn ($A$), on divise l'ensemble de ses chantillons sur $N$ ensembles, o $N$ est le nombre des classes.

$$CF = \frac{(\sum_{ij} A_{ij})^2}{\sum_j |A_j|} $$

O $\sum_{ij} A_{ij}$ est la somme de toutes les valeurs de toutes les classes, $\sum_j |A_j|$ est le nombre total des chantillons (de toutes les classes)

$$ TotalSS = \sum_{ij} A_{ij}^2 - CF$$

O $\sum_{ij} A_{ij}^2$ est les somme des carres de toutes les valeurs

$$ TreatmentSS = \sum_j \frac{(\sum_{i} A_{ij})^2}{|A_j|} - CF$$

O $\sum_{i} A_{ij}$ est la somme des valeurs de la classe $j$ et $|A_j|$ est le nombre des lments dans la classe $j$

$$ ErrorSS = TotalSS - TreatmentSS $$

# TODO complter cette fonction
def f_oneway(A): 
    # A est une liste de N listes o N est le nombre des classes
    # Ces listes n'ont pas la mme taille
    nbrs = np.array([len(a) for a in A]) # nombre des chantillons de chaque classe
    sommes = np.array([np.sum(a) for a in A]) # somme des valeurs de chaque classe
    sommes_caree = np.sum([np.sum(np.array(a)**2) for a in A]) # somme des valeurs carres de toutes les valeurs
    
    # Complter ici
    #====================================
    CF = (sommes.sum())**2/nbrs.sum()
    TotalSS = sommes_caree - CF
    TreatmentSS = np.sum(sommes**2/nbrs) - CF
    ErrorSS = TotalSS - TreatmentSS
    #====================================
    
    MSB = TreatmentSS / (len(A) - 1) 
    MSW =  ErrorSS/(nbrs.sum() - len(A))
    return MSB/MSW
